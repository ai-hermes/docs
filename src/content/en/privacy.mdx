**Effective Date: April 1, 2026**

Welcome to Rethink AI (hereinafter referred to as "the Application"). We take your privacy very seriously. This Privacy Policy explains how we collect, use, store, and protect your personal information.

**Please read this policy carefully. By using the Application, you agree to the terms of this Privacy Policy.**

---

## 1. Core Privacy Commitments

### 1.1 Chat Data Stays Local
**All your chat data — including contacts, group chats, and chat history — is stored exclusively on your local device. It is never uploaded to any cloud server or third-party service.**

### 1.2 Minimal Data Collection
We only collect essential anonymous behavioral analytics and error logs to improve the product experience. No chat content or personally identifiable information is involved.

---

## 2. Information We Collect

### 2.1 Chat Data (Stored Entirely on Your Device)
The Application accesses your chat data through local decryption or a Docker container, including:
- Contact information (username, nickname, remarks)
- Group chat information (group name, member list)
- Chat history (message content, timestamps, senders)

**Important:**
- This data is stored only in a local SQLite database on your device
- It is never uploaded to our servers or any third-party service
- You may delete the local database file at any time to erase all records

### 2.2 Account Information
To provide user authentication, we collect:
- Phone number (for login verification)
- User ID, Tenant ID (system-generated unique identifiers)

### 2.3 Device Information (Anonymous)
- Operating system type (Windows/macOS/Linux)
- System architecture (x64/arm64)
- Application version
- Language settings

### 2.4 Usage Analytics (Anonymous)
Through the PostHog analytics service, we collect:
- Application launch/close events
- Feature usage statistics (e.g., "clicked settings button", "switched model")
- Session duration
- Anonymous user ID (randomly generated UUID)

**We explicitly do NOT collect:**
- Chat message content
- Contact names or phone numbers
- Your conversations with the AI assistant
- Any personally identifiable sensitive information

### 2.5 Error Logs
When an application error occurs, we may collect:
- Error type and stack trace
- Application state at the time of error (excluding user data)
- Device environment information

---

## 3. How We Use Your Information

### 3.1 Providing AI Assistant Services
- **Local AI analysis**: Your chat data is processed locally to generate AI responses
- **LLM API calls**: When you actively ask the AI a question, only your input is sent to the LLM provider you have configured (e.g., OpenAI, Anthropic). We do not proactively upload your chat data

### 3.2 Product Improvement
- Analyzing anonymous usage data to optimize feature design
- Tracking error frequency to fix bugs

### 3.3 User Support
- Using your account information to provide technical support
- Verifying user identity to protect account security

---

## 4. Information Sharing and Disclosure

### 4.1 Third-Party Service Providers

#### PostHog (Analytics)
- **Purpose**: Anonymous behavioral statistics
- **Data type**: Application usage events, device information (no chat content)
- **Privacy control**: You can disable analytics in Settings
- **Data location**: PostHog servers (GDPR-compliant)

#### LLM Providers (OpenAI/Anthropic, etc.)
- **Purpose**: Processing AI questions you actively submit
- **Data type**: Only the question text you input
- **Important**: Do not include sensitive personal information in your questions
- **Privacy policy**: Subject to each LLM provider's own privacy policy

#### Portal Authentication Service
- **Purpose**: User login verification
- **Data type**: Phone number, verification code, User ID

### 4.2 Legal Requirements
We may disclose your information in the following circumstances:
- To comply with laws, regulations, court orders, or government requests
- To protect our legitimate interests or user safety
- To prevent fraud or security threats

### 4.3 No Sale of Data
**We will never sell, rent, or trade your personal information.**

---

## 5. Data Security

### 5.1 Technical Measures
- **Local encrypted storage**: Sensitive configurations (e.g., API keys) are encrypted using electron-store
- **Transmission encryption**: All network communications use HTTPS/gRPC encryption
- **Access control**: Only you can access the local database

### 5.2 Data Isolation
- Chat data runs in an isolated Docker container
- Data from different users is completely independent

### 5.3 Security Recommendations
- Keep your login credentials secure
- Regularly update the Application to the latest version
- Do not use the Application on public or shared devices

---

## 6. Your Rights

### 6.1 Access and Export
You may at any time:
- View all locally stored data (located in the application data directory)
- Export chat history and configuration files

### 6.2 Data Deletion
You have the right to delete:
- **Local chat data**: Delete the SQLite database file in the application data directory
- **Account data**: Use the "Delete Account" feature in Settings
- **Analytics data**: Contact us to delete anonymous records in PostHog

### 6.3 Disable Data Collection
In Settings, you can:
- Disable behavioral analytics (PostHog)
- Disable error log reporting
- Disable automatic update checks

---

## 7. Data Retention

### 7.1 Local Data
- **Chat data**: Retained permanently on your device until you manually delete it
- **Configuration files**: Automatically deleted when the Application is uninstalled

### 7.2 Cloud Data
- **Account information**: Deleted within 30 days after account deletion
- **Analytics data**: Retained for 90 days (anonymous data)
- **Error logs**: Automatically deleted after 180 days

## 8. Children's Privacy

The Application is not intended for children under the age of 14. If we discover that we have inadvertently collected personal information from a child, we will promptly delete it.

## 9. Cross-Border Data Transfers

- **Analytics data**: PostHog servers may be located outside your jurisdiction, but only anonymous data is transmitted
- **LLM API**: Depends on the provider you select (e.g., OpenAI is based in the United States)
