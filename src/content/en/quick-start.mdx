This guide will help you quickly install and start using the Rethink AI desktop application.

## Step 1: Download, Install, and Launch

Go to the [Download page](/en/download) to get the installer for your operating system:

- **macOS**: `.dmg` installer (supports Apple Silicon and Intel)
- **Windows**: `.exe` installer

Follow the system prompts to complete installation, then launch the app.

## Step 2: Configure Data Source

After launching, click the **Settings** button in the bottom-left corner to open the settings page:

![Click Settings in the bottom-left corner](/images/docs/main-settings.png)

### Configure Decryption

Select **Data Decryption** from the left sidebar in Settings, then follow the wizard steps:

![Decryption configuration wizard](/images/docs/settings-decryption.png)

1. **Environment Check** — Click "Check" and wait for the status to show passed
2. **Select User** — Choose the target user from the dropdown, then click "Confirm"
3. **Auto-Match Directory** — The system will automatically fill in the data directory path
4. **Get Data Key** — Click "Execute" and wait for the status to show passed
5. **Decrypt Database** — Click "Execute" and wait for decryption to complete
6. **Enable CDC Sync** (Optional) — Turn on to sync new messages in real time

Once all steps are complete, click the **Save** button in the bottom-right corner.

## Step 3: Mark Your Account

After decryption is complete, find and select your own account in the contact list to mark it as "Me":

![Select your account](/images/docs/select-user.png)

This helps the AI distinguish which messages are yours and which are from others when analyzing chat history, making responses more accurate.

## Start Using

Return to the main screen — you're ready to chat with the AI:

1. Type your question in natural language in the input box
2. Select an Agent mode: **Fast** (quick response) or **Deep** (deep reasoning)
3. The AI will automatically use tools to query and analyze your chat data

### Example Questions

- "Show me my recent chat list"
- "Find my chat history with John"
- "Search all messages mentioning 'project progress'"
- "Who sent me messages in the last week?"
