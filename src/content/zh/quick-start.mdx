本指南将帮助您安装、配置并开始使用 Rethink AI 桌面应用。

## 第一步：下载安装

从 [GitHub Releases](https://github.com/ai-hermes/wechat-mem0/releases) 下载适合您操作系统的安装包：

- **macOS**: `.dmg` 安装包 (支持 Apple Silicon 和 Intel)
- **Windows**: `.exe` NSIS 安装程序
- **Linux**: `.AppImage` / `.deb` / `.snap`

下载后按照系统提示完成安装。

## 第二步：配置 LLM 服务

启动应用后，点击左下角的 **设置** 按钮，配置您的 LLM 服务：

### 配置 API 密钥

1. 在设置面板中找到 **LLM 供应商** 配置区域
2. 选择一个 LLM 供应商 (如 OpenAI、DeepSeek 等)
3. 填入 API Base URL 和 API Key
4. 选择要使用的模型
5. 点击 **检测连通性** 确认配置正确

### 可选：配置 Tavily 搜索

如果希望 AI Agent 能够联网搜索，需要额外配置 Tavily API Key。

## 第三步：接入聊天数据

Rethink AI 提供两种方式访问您的聊天数据：

### 方式一：本地解密 (推荐)

适用于本机已安装 IM 客户端的场景：

1. 在设置面板中找到 **本地解密** 配置区域
2. 设置聊天数据库文件路径（通常会自动检测）
3. 点击 **启动解密服务**
4. 等待解密完成，即可开始查询

### 方式二：Docker 容器

适用于需要隔离运行 IM 的场景：

1. 确保已安装 Docker Desktop
2. 在设置面板中找到 **IM 运行时** 配置区域
3. 配置 Docker 镜像和端口
4. 点击 **启动容器**
5. 通过 VNC 连接 IM 客户端并登录

### CDC 实时同步 (可选)

启用 CDC (Change Data Capture) 后，应用会自动监测聊天数据库变更，实时增量解密新消息：

1. 在设置中启用 **CDC 同步**
2. 配置监控目录
3. 应用将自动检测并同步新数据

## 第四步：开始对话

一切配置完成后，您就可以开始与 AI 对话了：

1. 在主界面的输入框中输入您的问题
2. 选择 Agent 模式：**Deep** (深度推理) 或 **React** (快速响应)
3. AI 将自动调用数据查询工具，为您检索和分析聊天数据
4. 在消息气泡中可以查看 AI 使用的工具调用详情

### 示例问题

- "显示我最近的聊天列表"
- "帮我找一下和小明的聊天记录"
- "搜索所有提到'项目进度'的消息"
- "最近一周谁给我发过消息？"

## 常见问题

### 支持哪些操作系统？

支持 macOS (Intel/Apple Silicon)、Windows 和 Linux。

### 需要付费吗？

Rethink AI 应用本身免费开源。但您需要自行提供 LLM 服务的 API Key (如 OpenAI、DeepSeek 等)，这部分费用由 LLM 服务商收取。

### 聊天数据安全吗？

所有聊天数据仅存储在您的本地设备上，不会上传到任何服务器。仅在您主动向 AI 提问时，您输入的问题文本会发送到您配置的 LLM 服务商。

### 解密服务启动失败怎么办？

请确认 IM 客户端正在运行且已登录，然后检查数据库文件路径是否正确。可以在设置中查看日志获取详细错误信息。
