This guide will help you install, configure, and start using the Rethink AI desktop application.

## Step 1: Download and Install

Download the installer for your operating system from [GitHub Releases](https://github.com/ai-hermes/wechat-mem0/releases):

- **macOS**: `.dmg` installer (supports Apple Silicon and Intel)
- **Windows**: `.exe` NSIS installer
- **Linux**: `.AppImage` / `.deb` / `.snap`

Follow the system prompts to complete installation.

## Step 2: Configure LLM Service

After launching the app, click the **Settings** button in the bottom-left corner to configure your LLM service:

### Configure API Key

1. Find the **LLM Providers** section in the Settings panel
2. Select an LLM provider (e.g., OpenAI, DeepSeek)
3. Enter the API Base URL and API Key
4. Select the model to use
5. Click **Check Connectivity** to verify the configuration

### Optional: Configure Tavily Search

If you want the AI Agent to perform web searches, configure a Tavily API Key as well.

## Step 3: Connect Chat Data

Rethink AI offers two ways to access your chat data:

### Option A: Local Decryption (Recommended)

For when the IM client is already installed on your machine:

1. Find the **Local Decryption** section in Settings
2. Set the chat database file path (usually auto-detected)
3. Click **Start Decryption Service**
4. Wait for decryption to complete, then start querying

### Option B: Docker Container

For when you need the IM client running in an isolated environment:

1. Make sure Docker Desktop is installed
2. Find the **IM Runtime** section in Settings
3. Configure Docker image and ports
4. Click **Start Container**
5. Connect via VNC and log in to the IM client

### CDC Real-Time Sync (Optional)

Enable CDC (Change Data Capture) to automatically detect chat database changes and decrypt new messages in real time:

1. Enable **CDC Sync** in Settings
2. Configure the watch directory
3. The app will automatically detect and sync new data

## Step 4: Start Chatting

Once everything is configured, you can start chatting with the AI:

1. Type your question in the input box on the main screen
2. Select an Agent mode: **Deep** (deep reasoning) or **React** (quick response)
3. The AI will automatically use data query tools to search and analyze your data
4. Click on tool status indicators in messages to view call details

### Example Questions

- "Show me my recent chat list"
- "Find my chat history with Xiao Ming"
- "Search all messages mentioning 'project progress'"
- "Who sent me messages in the last week?"

## FAQ

### Which operating systems are supported?

macOS (Intel/Apple Silicon), Windows, and Linux.

### Is it free?

The Rethink AI application itself is free and open-source. However, you need to provide your own LLM API key (e.g., OpenAI, DeepSeek), and usage fees are charged by the LLM provider.

### Is my chat data secure?

All chat data is stored only on your local device and is never uploaded to any server. Only your question text is sent to the LLM provider when you actively ask a question.

### What if the decryption service fails to start?

Make sure the IM client is running and logged in, then verify the database file path is correct. Check the application logs in Settings for detailed error information.
