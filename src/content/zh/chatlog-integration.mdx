## 微信导出指南

### iOS 设备

1. 打开微信，进入目标聊天
2. 点击右上角 “...”，进入聊天详情
3. 向下滑动，找到“聊天记录”
4. 选择“导出聊天记录”
5. 选择时间范围
6. 选择“文本格式”
7. 通过“文件”应用保存到本地

### Android 设备

1. 打开微信，进入目标聊天
2. 点击右上角 “...”
3. 选择“查找聊天记录”
4. 点击右上角“导出”
5. 选择保存位置

## WhatsApp 导出指南

### 导出步骤

1. 打开 WhatsApp
2. 进入目标聊天
3. 点击联系人名称或群组名称
4. 向下滑动找到“导出聊天”
5. 选择“不包含媒体文件”
6. 选择保存或分享方式

### 注意事项

- 导出的文件包含所有聊天记录
- 建议选择“不包含媒体”以减小文件大小
- 文件格式为 `.txt`

## Telegram 导出指南

### 使用 Telegram Desktop

1. 下载并登录 Telegram Desktop
2. 打开目标聊天
3. 点击右上角三个点
4. 选择“导出聊天记录”
5. 在设置中：
   - 格式选择 “JSON” 或 “HTML”，推荐 JSON
   - 取消勾选媒体文件
   - 设置时间范围
6. 点击“导出”

## 文件格式要求

### TXT 格式

```text
[2024-01-15 10:30:25] 张三: 你好，最近怎么样？
[2024-01-15 10:31:02] 李四: 挺好的，谢谢关心！
[2024-01-15 10:32:15] 张三: 周末有空吗？想约你吃饭
```

### JSON 格式

```json
{
  "messages": [
    {
      "date": "2024-01-15T10:30:25",
      "from": "张三",
      "text": "你好，最近怎么样？"
    },
    {
      "date": "2024-01-15T10:31:02",
      "from": "李四",
      "text": "挺好的，谢谢关心！"
    }
  ]
}
```

## 常见问题

### 导出的文件太大怎么办？

建议选择特定的时间范围导出，或者排除媒体文件。Rethink AI 支持最大 50MB 的文件上传。

### 文件编码问题

请确保文件使用 UTF-8 编码。如果出现乱码，可以使用文本编辑器将文件另存为 UTF-8 格式。

### 隐私内容如何处理？

您可以在上传前手动删除敏感信息。我们的系统不会存储您的原始聊天记录。
